## Mobile Sales Dashboard (Power BI Project)

## 🔹 Overview
This project is an interactive Power BI dashboard built to analyze mobile sales performance across different dimensions such as brand, city, payment method, and customer ratings.


## 🔹 Key Features

- KPI Cards (Total Sales, Quantity, Transactions, Avg Price)
- Sales by Brand and Mobile Model
- City-wise Sales (Map Visualization)
- Payment Method Analysis
- Monthly Sales Trend
- Customer Rating Distribution
- Interactive Filters (Brand, Model, Payment, Month)

## 🔹 Tools Used

- Power BI
- DAX (Basic Calculations)
- Data Visualization Techniques
  
## 🔹 Key Insights

- A few mobile models contribute significantly to total sales
- Certain cities have higher sales concentration
- Majority of customer ratings are positive
- Sales trend remains stable across months
